THE EXCEPTIONS

Despite the fact that many Domain-Specific Languages are not Turing Complete, there do exist Domain-Specific Languages that are actually Turing Complete, and can in fact compute pretty much anything. Ofcourse, this is not to say that these languages are good at it. Instead, this only suggests its capability, where memory permitting, any program written in it can be solved by any other deterministic language.

Take for example, MATLAB fulfils the criteria of being a Turing Complete Language by being able to implement control flow using 'If' statements. MATLAB even comes with a lot of features that modern programming languages such as Java and Python possess.

Yet, MATLAB is still able to achieve its goals as a Domain Specific Language, by simplifying the syntax required in solving problems that deal with matrix manipulation and linear algebra, thereby making it much more accessible to users who have little to no experience in programming.
